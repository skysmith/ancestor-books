# Thumbnail Sheet

## Purpose

Composition guide for Book 5's suspense-comedy pacing. This solves eye-lines, object clarity, and text-safe space before detailed art.

## Global Composition Rules

- Keep the camp readable and spatially clear.
- Let props do a lot of the work: chest, cask, handkerchief, honey.
- Build suspense through where the Tory looks.
- Let relief arrive late.

## Value Strategy

- Opening: pale morning with emotional tightness
- Middle: warmer honey accents against dull camp colors
- Ending: softened values and shared release
