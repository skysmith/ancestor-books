# Anchor Regeneration Notes

## Problem In First Pass

The initial anchor images were too visually similar:

- repeated Daniel foreground portrait framing
- repeated blue/orange palette balance
- repeated medium-close camera distance
- repeated emotional expression pattern

That created consistency, but not enough storytelling variety.

## Regeneration Strategy

Each anchor spread now has a different visual job:

1. `spread-01`
wide establishing image, environment-led

2. `spread-04`
object-led hardship image, shoe and fire as focal point

3. `spread-06`
group tableau, captain-centered

4. `spread-08`
split spatial composition, Daniel separated from retreating figures

5. `spread-10`
near-silhouette guard image, distant and watchful

6. `spread-14`
minimal ending landscape, tiny figures or near-empty field

## Constraint Summary

- fewer Daniel close-ups
- more environment-led spreads
- stronger variation in camera distance
- stronger variation in focal subject
- stronger variation in value structure
- brighter readable night scenes using moonlight, snow bounce, or distant lantern light
