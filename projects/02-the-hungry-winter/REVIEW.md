## Working Method

1. Read the spread text in `manuscript/text-only-layout.md`
2. Open the current spread in `storyboard/hybrid-frames-v2/`
3. Generate or edit candidate images
4. Save experiments in `storyboard/renders/raw/`
5. When one wins, copy it into `storyboard/renders/selects/` as `spread-XX-selected.png`
6. Rebuild the hybrid dummy after any meaningful change set

## Current Lead Notes

### Anchor Set

The current `v2` anchor set is the active baseline because it fixes the biggest first-pass problem: too many similar Daniel-centered images.

What the current anchor set does better:

- `spread-01`: environment-led opening instead of another portrait
- `spread-04`: object-led shoe and fire hardship image
- `spread-06`: captain-centered group tableau
- `spread-08`: more spatial separation between Daniel and the mission figures
- `spread-10`: quieter, more silhouette-based guard image
- `spread-14`: simpler endurance ending

Global direction to preserve:

- colder than the other books
- readable night scenes, not muddy darks
- less portrait repetition
- hardship conveyed through environment, posture, and repetition
